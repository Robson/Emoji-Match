Emoji Match

This is a game for 2+ players,
where one person secretly chooses an emoji, then everyone else works together to figure out what it is.

If you are playing on a phone/tablet, it is recommended to use landscape mode.

Start by choosing one person to be Player 1. Everyone else is a Guesser.
